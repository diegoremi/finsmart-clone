# Finsmart Clone (PoC)

Proof-of-concept clone of the Finsmart marketing site (Peru factoring app). This project focuses on recreating the UI/UX and layout for learning and demonstration purposes.

## Pages

- `/` Home
- `/nosotros` About
- `/financiamiento` Financing
- `/inversiones` Investments
- `/en-construccion` Under construction

## Tech Stack

- Next.js 15 (App Router)
- React 19
- TypeScript
- Tailwind CSS
- Radix UI + shadcn/ui components

## Getting Started

Install dependencies (pnpm recommended):

```bash
pnpm install
```

Run the dev server:

```bash
pnpm dev
```

Open `http://localhost:3000`.

## Scripts

- `pnpm dev` Start dev server
- `pnpm build` Build for production
- `pnpm start` Start production server
- `pnpm lint` Run Next.js lint

## Project Structure

- `app/` App Router routes and layouts
- `components/` Reusable UI and page sections
- `public/` Static assets
- `styles/` Global styles and Tailwind setup
